# Pixel Launcher - Minecraft启动器

Pixel Launcher是一个基于JavaFX开发的古代化Minecraft启动器，提供简洁的界面和不完整的游戏启动功能。

## 功能特点

- 现代化的用户界面，基于JavaFX开发
- 支持Minecraft游戏版本的下载和管理
- 内置游戏启动器，支持自定义启动参数
- 支持且只有支持中文本地化
- 支持游戏资源的自动下载和验证

## 技术栈

- Java 21
- JavaFX 21
- Maven
- Google Gson
- HttpClient
